Control Flow And Loops

Control flow is the order in which a computer executes lines of code in a script. Code is run from the first line to the last in order, that is only if no condition statements, loops, or functions are written to name a few. Loops will repeat a block of code until a certain condition is true. Conditional statements are like a fork in the road, the computer will only go down the path in which it is satisfied with and functions are a block of code that will only run when or if called.
An example of control flow and loops in everyday life would be doing the dishes. Your day will be flowing from one thing to the next until you decide to do the dishes:
While all the dishes aren't clean repeat these steps{ clean next dish }
Once the dishes are complete you are now free for your next task.

DOM interaction

The DOM (Document Object Model) is the model in which objects are manipulated on an HTML document. HTML documents need to be able to be manipulated by the user in order to have an interactive web page. The DOM represents elements in a HTML document as nodes, that way we can use a programming language (most likely Javascript) to grab those nodes, alter them, then re-enter them into the HTML document, changing the web page in real time.

Objects vs Arrays

Objects are a data type in most programming languages which store data in a key value pair. To access a value in an object, you first need to write the object variable followed by either a dot then the key to the value you want or follow the object variable with two square brackets and a string of the key to the value you want inside it.


Constructing OBJECT: let object = {key : value} || Accessing Value: let get-value = object.key or object['key']


Arrays are very different, they store data in a container in order starting from 0, 1, 2... until the last item. You can access an item by writing the object followed by square brackets with the index of the item you want between them. This can be hard to keep track of when the array becomes large and most of the time the datas not sorted, so a lot of the time you use a for loop which loops through the array accessing one piece of data at a time and if you found an item you wish to use, the computer will execute a different block of code.


Constructing ARRAY: let array = [0, 1, 2, 3, 4] || Accessing ARRAY: let first-number = array[0]

Functions

A function is a block of code which the computer will not execute unless called upon, which has many benefits. Rather than writing the same thing over and over again, you can write the block of code out once and just call that function when needed. Another benefit would be if you only wanted a block of code to be executed when a certain event has happened whether that be user input, timed, HTTP response etc.
